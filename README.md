# EventEase Blazor Application

## Project Overview

EventEase is a Blazor web application developed to manage events and registrations. This project demonstrates the use of Microsoft Copilot to generate code, debug issues, and optimize application performance.

The application includes routing, validation, state management, and an attendance tracking system.

## 2. Event Card Component with Two-Way Data Binding (5 pts)

An Event Card component was created using Microsoft Copilot.

Fields included:

* Event Name
* Event Date
* Event Location
* Event Description

Two-way data binding was implemented using:

@bind

This allows user input to automatically update the UI.

Copilot assisted by:

* Generating the component structure
* Suggesting form layout
* Providing binding syntax
